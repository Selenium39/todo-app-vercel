
# 待办事项应用

这是一个使用 Next.js 和 Supabase 构建的简单待办事项应用。

## 功能

- 新增待办事项  
- 标记待办事项为已完成
- 删除待办事项
- 按天分组归档已完成的待办事项
- Supabase 后端储存数据

## 使用技术

- Next.js - React 框架
- Chakra UI - UI 组件库
- Supabase - 后端数据库
- Day.js - 处理时间显示
- React Markdown - 渲染 Markdown 语法
